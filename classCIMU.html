void CIMU::Update ( const CVect3 *  pwm,
const CVect3 *  pvm,
int  nSamples 
)

Update IMU observations using Multi-subsample algorithm.

When nSamples == 1, onePlusPre is true, IMU will update using one-plus-previous sample, onePlusPre is false, IMU will update without compensation.

Parameters
[in]pwmAttitude increment(Gryo measurements)
[in]pvmVelocity increment(Accelerometer measurements)
[in]nSamplesNumber of subsample( 1:One-plus-previous sample or without compensation >1: Muti-subsample,max subsample is 5)
